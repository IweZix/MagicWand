# Création d'un item
- créer une classe qui hérite de Item dans mymod/item/custom
- enregistrer l'item dans MyModItem.java
- ajouter la texture dans resources/assets/mymod/textures/item
- associer la texture à l'item dans resources/assets/mymod/models/item
- ajouter la traduction dans resources/assets/mymod/lang
- ajouter la receipe dans resources/assets/mymod/recipes
